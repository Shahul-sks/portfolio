This page contains the case study of Dr.Appointment Website which includes the Project Overview, Tools Used and Live Links to the official product.

Project Overview

Dr.Appointment is a web page through which patient can book an online appointment with a doctor. Once the doctor approved the appointment then patient can procced with a online payment and confirm his appointment. Dr. Apoinntment will help in generating Prescription online and will be shared with the patient. Patient can also keep track of Height, weight, BMI, BP etc.

I took User Interface ideas by visiting different types of websites of the same industry and visited platforms like dribbble to get inspiration for building a good User Interface.
